## 任务2：城市天际线效果
### 实现方案
- 创建了skyline.html文件
- 使用纯CSS实现视觉效果，无需JavaScript

### 技术细节
1. 布局设计
   - 使用flex布局排列大楼
   - 10栋不同高度的大楼（280px-450px）
   - 每栋大楼宽度40px

2. 视觉效果
   - 背景：深蓝色渐变模拟夜空
   - 大楼：深色剪影效果
   - 窗户：黄色灯光效果（opacity: 0.3-0.5）
   - 动画：窗户灯光闪烁效果（4秒循环）

### CSS 关键技术
```css
// 背景渐变
background: linear-gradient(to bottom, #1a1a2e 0%, #16213e 50%, #0f3460 100%);

// 窗户动画
@keyframes flicker {
    0%, 100% { opacity: 0.3; }
    50% { opacity: 0.5; }
}
```
